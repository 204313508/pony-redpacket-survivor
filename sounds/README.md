# 音效文件说明

游戏需要以下音效文件，请将对应的音频文件放在此目录中：

## 必需的音效文件

1. **attack.mp3** - 攻击音效
   - 当玩家点击鼠标左键攻击时播放
   - 建议时长：0.1-0.2秒
   - 建议：快速、有冲击力的音效

2. **monster_death.mp3** - 怪物死亡音效
   - 当怪物被击败时播放
   - 建议时长：0.2-0.3秒
   - 建议：清脆、令人愉悦的音效

3. **collect.mp3** - 收集红包音效
   - 当玩家收集红包时播放
   - 建议时长：0.05-0.1秒
   - 建议：清脆、高音调的音效

4. **upgrade.mp3** - 升级音效
   - 当玩家升级时播放
   - 建议时长：0.3-0.5秒
   - 建议：欢快、有成就感的音效

## 音频格式要求

- **格式**：MP3、WAV、OGG 等浏览器支持的格式
- **音量**：适中，不要过大
- **质量**：建议使用高质量的音频文件

## 注意事项

- 所有音效文件必须放在 `sounds/` 目录下
- 文件名必须与上述列表完全一致
- 如果没有音效文件，游戏将不会播放声音但可以正常运行

## 推荐音效来源

你可以从以下网站获取免费音效：
- Freesound.org
- Mixkit.co
- Pixabay.com
- Zapsplat.com

请确保遵守音效文件的使用许可协议。
